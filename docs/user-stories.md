Coach/Admin
-
  - As a coach, I want to be able to create game events and add information about it (location, kickoff time, etc)
  - As a coach, I want to be able to edit events, their information, and delete them.
  - As a coach, I want to add RSVP deadlines to the events I created
  - As a coach, I want to be able to see attendance to my events as it stands
  - As a coach, I want to be able to invite players and parents to my team's chat
  - As a coach, I want to be able to message all my team

Parent
-
  - As a Parent, I want to be able to sign in as myself or on behalf on my child and myself
  - As a Parent, I want to be able to see the team schedule so I can plan ahead
  - As a Parent, I want to be able to chat with the coach (and player if signed up) in private chat
  - As a Parent, I want to be able to RSV to scheduled events so coach knows my player's attendance
  - As a Parent, I want to be able to receive notifications 2 hours prior to RSVP deadline
  - As a Parent, I want to be able to message in my team chat

Player
-
  - As a Player, I want to be able to sign in as myself and link to my Parent
  - As a Player, I want to be able to see the team schedule so I can plan ahead
  - As a Player, I want to be able to chat with the coach and parent privately
  - As a Player, I want to be able to RSV to scheduled events so coach knows my attendance
  - As a Player, I want to be able to receive notifications 2 hours prior to RSVP deadline
  - As a Player, I want to be able to message in my team chat
